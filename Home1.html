Welcome!

  This is a website where you will learn how historical events influence architecture. We will be focusing on architecture and events in Europe and American from the 1400s to modern time. This website is aimed at teens 11-18 so if you're a teen you're in the right place! There are five main sections to this lesson. There is a small quiz at the end of every section and a large quiz at the end. Have fun!
